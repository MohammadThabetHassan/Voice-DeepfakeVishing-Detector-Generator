Voice Deepfake Vishing Detector & Generator

1. Detect Deepfake Audio

Upload a WAV file or use your microphone to record a few seconds of speech, then click Detect.

Note: Many browsers record in WebM by default, which this demo server cannot decode. For real-time demos, transcode on the server with FFmpeg or pydub.

You must use the form below to upload a WAV file; URL/query-string detection isn’t supported.

2. Generate Deepfake Audio

Provide a source WAV file and some text. The server will generate a pitch-shifted “deepfake” clip.
