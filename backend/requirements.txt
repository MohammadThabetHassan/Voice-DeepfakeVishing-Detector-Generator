# ── TTS: XTTS v2 (RECOMMENDED for voice cloning — pip installable) ─────────
# XTTS v2 is the easiest way to get high-quality voice cloning:
#   pip install TTS>=0.22.0
#
# First run will auto-download the model (~2 GB) to ~/.local/share/tts/
# Supports languages: en, es, fr, de, it, pt, pl, tr, ru, nl, cs, ar, zh, ja, hu, ko
#
# Hardware:
#   CPU:  Works but slower (10–30 s per synthesis)
#   GPU:  Recommended (NVIDIA, CUDA 11.8+, ~4 GB VRAM)
#         The model will auto-detect and use GPU if available.
# TTS>=0.22.0

# ── TTS: IndexTTS2 (alternative voice cloning, source install) ─────────────
# IndexTTS2 is NOT a pip package — it must be installed from source:
#
#   Step 1: Clone the repo
#     git clone https://github.com/index-tts/index-tts.git /opt/index-tts
#     cd /opt/index-tts
#
#   Step 2: Install with uv (required — pip is NOT supported by IndexTTS2)
#     pip install uv
#     uv sync --all-extras
#
#   Step 3: Download model weights (~4 GB)
#     uv tool install "huggingface-hub[cli]"
#     huggingface-cli download IndexTeam/IndexTTS-2 --local-dir checkpoints
#
#   Step 4: Tell this backend where IndexTTS2 lives
#     export INDEXTTS_DIR=/opt/index-tts
#
#   Step 5: Start the backend
#     cd /path/to/this/project/backend
#     uvicorn app:app --reload
#
# Hardware:
#   CPU:  Works but slow (30–120 s per synthesis on a modern CPU)
#   GPU:  Recommended (NVIDIA, CUDA 12.8+, ~4 GB VRAM for FP16)
#         Add --use_fp16 flag in app.py IndexTTS2() constructor for GPU mode.
